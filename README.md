# NexaCode NestJs Template

## Description
* Nexacode NestJs template
* 기술스택
  * Node Version Control : NPM
  * Language: TypeScript
  * Framework: NestJs
  * ORM : TypeORM
  * DB: mysql

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
